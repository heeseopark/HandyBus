## HandyBus 버스 대절 플랫폼
> 5개의 콘서트, 1,700만원의 매출 (MVP 테스트)
지방에서 수도권 콘서트를 가고자하는 이용자들을 위한 서비스입니다. 이를 위해 콘서트를 등록하고 예약을 모집하는 사이트를 만들었습니다.

## 목차
- 프로젝트 소개
- 기술 스택
- ERD

## 프로젝트 소개
고려대학교 소프트웨어 공학 팀 프로젝트입니다. 팀 내 Backend 개발을 맡아 진행했습니다. (Backend, Frontend, Designer 팀 내 보유)
